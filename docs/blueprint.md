## Core Features:

- Hero Section: Displays a welcoming photo and introductory text.
- Course Information: Showcases violin and alto courses in Lyon with an inviting image.
- Age-Based Classes: Highlights options for both individual and group music lessons for various age ranges, accompanied by relevant imagery.
- Suzuki Method Intro: Details the Suzuki teaching method for children's violin lessons.
- Inspirational Learning: Showcases the inspiration behind learning violin in childhood.
- Location Map: Displays stylized location of the music school with contact details.
- Activity Carousel: Includes a looping image slider that showcases activity photos related to music classes and other relevant moments. Transitions should be slow and infinite.

## Style Guidelines:

- Primary color: Soft Cream/Beige (#F9F7F2) for the main background, offering a gentle and inviting feel.
- Secondary color: Pale Apricot (#FDECE3) for alternating sections, adding warmth and subtle visual variation.
- Tertiary color: Gentle Sage (#E6EBE8) to further alternate backgrounds, maintaining a calm atmosphere.
- Accent color: Muted Terracotta (#C58E70) used for buttons, titles, and highlights, providing contrast.
- Headline font: 'Playfair' serif typeface for titles, lending an elegant touch.
- Body font: 'Nunito' sans-serif typeface for body text, ensuring clean readability. Note: currently only Google Fonts are supported.
- Minimalist layout with generous whitespace to promote clarity and calm.
- Use gentle, organic curves or wave-shaped SVG separators between sections to create smooth transitions between background colors.
- Implement fade-in and slide-up effects with smooth and slow animations when scrolling to make the site feel dynamic but not overwhelming.
- Use simple, hand-drawn style icons
